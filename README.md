Password Strength Checker 

This is a simple Python project that checks whether a password is strong or weak.

Features
- Checks minimum length (8 characters)
- Verifies if password contains numbers
- Verifies if password contains special characters

How it works
The user enters a password, and the program evaluates it based on predefined conditions.

Output
- Prints "strong" if conditions are satisfied
- Otherwise prints "weak"

Tech Used
- Python (Basic concepts: loops, conditions, strings)

Future Improvements
- Add uppercase/lowercase checks
- Add real-time feedback
- Convert into GUI/Web app
